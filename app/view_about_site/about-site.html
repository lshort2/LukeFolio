What is the purpose of this site?

This site is intended to show different involvements in my life. I use the vague word "involvements" intentionally. Almost all the projects I have worked on are viewable from GitHub, so I am not so sure there would be much added value to listing them all out here. This site is more so a way to share mention of some overarching things in my life that have probably spanned months or even years.

It is not, however, intended to be a full-fledged life story or documentary. This is a way to show others some things that I have been involved with and (hopefully) spark some new questions.

This also served as an excuse to use a few tools I have been waiting to try, like Bower.
